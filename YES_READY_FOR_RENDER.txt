## 🚀 HOW TO SEE YOUR APP ON RENDER (5 MINUTES)

### Step 1: Create Database on Render
```
1. Visit: https://dashboard.render.com
2. Click "New +" → "PostgreSQL"
3. Name: aptitude-hub-db
4. Click "Create"
5. Copy the Connection String
```

### Step 2: Deploy Web Service
```
1. Click "New +" → "Web Service"
2. Connect GitHub repo: chandrabhanu35/web-app
3. Set build: npm install
4. Set start: npm start
5. Add environment variables:
   - DATABASE_URL = (from database)
   - JWT_SECRET = your_secret_key
   - NODE_ENV = production
6. Click "Create Web Service"
7. Wait 3-5 minutes...
```

### Step 3: See Your Live App
```
Your app will be at:
🌐 https://aptitude-hub-web.onrender.com

View it, test it, share with students!
```
